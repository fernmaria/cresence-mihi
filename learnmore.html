Cres:Mi Services

Calendar

The Cres:Mi Student Assisting Website provides a calendar wherein it can function as a planner. The user can easily navigate through the calendar with the arrows or use the search bar to look for a specific date from before or the future.

To Do List

The website provides a to-do list where a student can indicate how long they want to work on a project. The schedule will be saved in their device even if they close the website.

Review Quizzes

The Cres:Mi Student Assisting Website provides numerous quizzes revolving the Web Development Subject and aims to create quizzes on other subjects in the future. The user can go back to the review quizzes as much as they want to test their memory.
